## Что ожидать

- **Qt приложения предпочтительнее GTK**. Последние по-прежнему оформлены в соответствующем стиле, но не используются по умолчанию.
- Оформление **Material Design 3**
- **Знакомые сочетания клавиш** для тех, кто перешел с Windows или Gnome. Никаких странных сочетаний клавиш из Vim!
- **Уверенностe в своём мнении (Opinionatedness)**: Это само собой разумеется для dotfiles.
